## âš™ï¸ How Pair Code Works (Urdu + English)

1. Jab WhatsApp bot login maangega, Baileys library ek **pairing code** ya **QR string** degi.  
2. Ye bot us code ko `/pairtext` endpoint par save karta hai.  
3. Aap browser me ja kar `https://<your-app>.herokuapp.com/pairtext` open karo.  
4. Wahan pe JSON me pairing code mil jayega.  
5. Ya agar aapke paas `public/pair.html` hai to us page pe code automatically show ho jayega.

> **Note:** Pairing ke liye doosra mobile phone use karein (same phone se scan mat karein).
